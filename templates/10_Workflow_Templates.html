⚡ Max Gym — Workflow Templates

13 automations for GHL. Configure these in Automations → Workflows. All labels in Spanish. Pipeline stage: "Tour / Visit Scheduled".

📋 Pipeline Stage Mapping (Max Gym)

  • New Lead — Any inbound lead (WhatsApp, IG, Meta Ads, Google, referral)
  • Tour / Visit Scheduled — Tour booked via bot
  • Prospect — Tour completed, interested, human follows up
  • Member Signed — Agreement + payment completed
  • Active Member — Member active 30+ days
  • Lost — Human-marked as lost (not DQ)
  • Internal: Disqualified (via bot), Cold (inactivity), Won
WF 1 Actualizar estado negocio cuando IA agenda tour
When the bot successfully books a tour, move the opportunity forward
Trigger: Bot Action "Mover a Oportunidad" Pipeline: Membership / Subscription Services
[TRIGGER] Bot Action: Mover a Oportunidad │ ▼ [ACTION] Update Contact Field │ Field: "Bot Status" → "Meeting Booked" │ ▼ [ACTION] Add Tags │ Add: meeting-scheduled, ia-qualified │ Remove: ia-qualifying, new-lead │ ▼ [ACTION] Update Opportunity Stage │ Pipeline → Stage: "Tour / Visit Scheduled" │ ▼ [ACTION] Send Internal Notification │ Channel: Slack/Email → Dorayme │ Message: "{{contact.name}} agendó un tour vía IA"
Max Gym customization: Stage changed to Tour / Visit Scheduled. Tags updated to match Max Gym tag library. Notify Dorayme.
WF 2 Actualizar estado negocio cuando IA descalifica
When the bot disqualifies a lead (TIPO A–F), move to disqualified with reason
Trigger: Bot Action "Descalificar Lead" 6 DQ Types: A (budget), B (location), C (no decision), D (out of range), E (committed), F (underage)
[TRIGGER] Bot Action: Descalificar Lead │ ▼ [CONDITION] What disqualification type? │ ├── TIPO A → tag: dq-budget-too-low │ ├── TIPO B → tag: dq-wrong-profile │ ├── TIPO C → tag: dq-no-decision-maker │ ├── TIPO D → tag: dq-out-of-range │ ├── TIPO E → tag: dq-already-committed │ └── TIPO F → tag: dq-underage │ ▼ [ACTION] Add Tags │ Add: ia-disqualified, dq-[type], stop-bot │ ▼ [ACTION] Update Opportunity Stage │ Pipeline → Stage: "Lost" (Disqualified) │ ▼ [ACTION] Update Contact Field │ Field: "Disqualification Reason" → "[TIPO X - Description]" │ Field: "Bot Status" → "Disqualified" │ ▼ [ACTION] Stop Bot for this contact
Max Gym customization: 6 DQ types (A–F). New TIPO F = dq-underage (edad mínima 13 años). STOPBOT_DESCALIFICADO message sent before workflow fires.
WF 3 Actualizar estado negocios descalificado (manual)
Secondary handling when a lead is manually marked as disqualified by staff
Trigger: Opportunity stage changes to "Lost" + manual DQ
[TRIGGER] Opportunity Stage Changed → "Lost" │ ▼ [CONDITION] If stage reason = "Disqualified" │ ▼ [ACTION] Add Tags │ Add: disqualified, lost │ ▼ [ACTION] Update Contact Field │ Field: "Bot Status" → "Disqualified"
Max Gym customization: Manual DQ by staff. Checks for ia-disqualified tag presence.
WF 4 Envío de info a píxel por descalificados
Send conversion event to ad platform when a lead is disqualified (for audience exclusion)
Trigger: Opportunity stage changes to "Lost" with disqualified tag
[TRIGGER] Opportunity Stage Changed → "Lost" │ ▼ [CONDITION] Tag contains "ia-disqualified" │ ▼ [ACTION] Send Conversion Event (Google Ads / Facebook) │ Event: "Disqualified Lead" │ Value: 0 │ Currency: USD
Max Gym customization: Ensures ad platforms don't retarget disqualified leads. Protects ad spend efficiency.
WF 5 Actualizar estado negocios fríos
Automatically move leads to "Cold" after 14 days of inactivity
Trigger: Scheduled — Daily at 8:00 AM
[TRIGGER] Scheduled: Daily at 8:00 AM │ ▼ [CONDITION] Contact has NO activity in 14 days │ AND opportunity stage = "New Lead" or "Tour / Visit Scheduled" │ AND tag "cold" is NOT present │ ▼ [ACTION] Add Tags │ Add: cold, nurturing │ ▼ [ACTION] Update Opportunity Stage │ Stage: Cold │ ▼ [ACTION] Start Workflow 11 (Follow-up after 72h sequence)
Max Gym customization: Uses Max Gym stages (New Lead, Tour / Visit Scheduled). Tags: cold, nurturing.

WF 7 Actualizar estado negocios perdidos
Handle lost deals gracefully — capture reason, tag, and move to nurture if appropriate
Trigger: Opportunity stage changes to "Lost" (NOT via disqualified)
[TRIGGER] Opportunity Stage Changed → "Lost" │ ▼ [CONDITION] Tag "ia-disqualified" is NOT present │ ▼ [ACTION] Add Tags │ Add: lost │ ▼ [ACTION] Update Contact Field │ Field: "Loss Reason" → from human input │ ▼ [ACTION] Add to Nurture Sequence │ Start Workflow 11 if salvageable
Max Gym customization: "Lost" stage used. lost tag added. Human enters loss reason.
WF 8 Duplicar info de contacto a oportunidad
Automatically copy key contact fields to the opportunity when created
Trigger: Opportunity created
[TRIGGER] Opportunity Created │ ▼ [ACTION] Copy Contact Field to Opportunity │ {{contact.name}} → {{opportunity.name}} │ {{contact.email}} → {{opportunity.email}} │ {{contact.phone}} → {{opportunity.phone}} │ {{contact.objetivo_principal}} → custom field │ {{contact.situacion_actual}} → custom field │ {{contact.presupuesto}} → custom field │ {{contact.edad}} → custom field │ {{contact.tipo_entrenador}} → custom field │ ▼ [ACTION] Add Opportunity Tag │ Add: bot-created (if from bot)
Max Gym customization: Copies Max Gym qualification fields (objetivo_principal, situacion_actual, presupuesto, edad, tipo_entrenador) to opportunity for human sales visibility.
WF 9 Recordatorio de Tour (WhatsApp)
Send meeting reminder via WhatsApp before scheduled tour
Trigger: Calendar event — 24h before AND 1h before
[TRIGGER] Calendar Event: 24 hours before │ ▼ [CONDITION] Appointment status = "Scheduled" │ ▼ [ACTION] Send WhatsApp Template │ Template: recordatorio_visita_24h │ {{1}} = {{contact.first_name}} │ {{2}} = {{appointment.start_time}} │ ▼ [ACTION] Add Tags │ Add: meeting-reminder-sent
[ALTERNATE TRIGGER] Calendar Event: 1 hour before │ Template: recordatorio_visita_1h │ Same variables as 24h
Max Gym customization: Uses Max Gym WhatsApp templates (recordatorio_visita_24h, recordatorio_visita_1h). Calendar event = Tour / Visit Scheduled.
WF 10 Desactivar IA cuando un lead se marque como descalificado
When a lead is disqualified (by bot or human), stop the bot from engaging further
Trigger: Tag "ia-disqualified" added OR stage = "Lost" with DQ reason
[TRIGGER] Tag Added: "ia-disqualified" │ ▼ [ACTION] Stop AI Agent Bot for this contact │ Setting: Bot deactivated │ ▼ [ACTION] Add Tags │ Add: stop-bot │ ▼ [ACTION] Update Contact Field │ Field: "Bot Status" → "Deactivated"
Max Gym customization: Uses Max Gym's ia-disqualified and stop-bot tags. Bot Status field tracks deactivation.
WF 11 Seguimiento leads — secuencia de re-engagement
Re-engage leads who haven't responded after 72 hours, with follow-up touches
Trigger: Time-based — 72h since last message
[TRIGGER] Time-based: 72 hours since last message │ ▼ [CONDITION] Contact is in "New Lead" or "Tour / Visit Scheduled" stage │ AND "stop-bot" tag NOT present │ AND "human-handover" tag NOT present │ ▼ [ACTION] Send WhatsApp — seguimiento_72_horas │ Variable: {{contact.first_name}} │ ▼ [ACTION] Wait 5 days │ ▼ [ACTION] Send WhatsApp — seguimiento_5_dias │ ▼ [ACTION] Wait 7 days │ ▼ [ACTION] Send WhatsApp — seguimiento_15_dias │ ▼ [CONDITION] If no response after all follow-ups: │ Tags: cold, no-reply │ Move to Cold / Lost
Max Gym customization: Sequence uses Max Gym WhatsApp templates. Stages: New Lead, Tour / Visit Scheduled. Tags: cold, no-reply.
WF 12 Creación de negocio leads entrantes
When a new lead comes in (form, website, manual), create contact + opportunity + start bot
Trigger: Form submitted OR new contact created with source
[TRIGGER] Form Submitted OR Contact Created │ ▼ [CONDITION] Contact source = "Landing Page" OR "Website" OR "Form" │ ▼ [ACTION] Create/Update Contact │ Fields: name, email, phone, objetivo_principal, edad │ ▼ [ACTION] Add Tags │ Add: new-lead, source-[source], ia-qualifying │ ▼ [ACTION] Create Opportunity │ Pipeline: Membership / Subscription Services │ Stage: "New Lead" │ ▼ [ACTION] Start AI Agent Conversation │ Bot: Max Gym Sales Bot │ Trigger: Automatic first message (Phase 0 greeting) │ ▼ [ACTION] Send Internal Notification │ Message: "Nuevo lead: {{contact.name}}"
Max Gym customization: Pipeline = Membership / Subscription Services. Stage = New Lead. Bot = Max Gym Sales Bot with Phase 0 greeting. Source tags: source-whatsapp, source-instagram, source-website, etc.

Max Gym customization: Stage "Won" → Member Signed. Tags: google-ads, source-paid. Requires Google Ads conversion tracking setup.
